## 正确打开方式
hug2、hug3 运行必须依赖同文件夹下的 hug.json 剧情文件，请先将程序与剧情文件放置在同一个文件夹，然后再运行。

程序会要求输入**花名册中存在**的同学姓名缩写，建议您对照 hug.json 输入。
