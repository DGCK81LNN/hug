# 抱一抱

由一个 Python turtle 练习程序衍生出的一系列作品

**hug** by [RichardM / 天府灵山行者](https://space.bilibili.com/300711293) （孟）
**hug2**, **hug3** by [RoyL / DGCK81LNN](https://space.bilibili.com/328066747) （罗）

## 正确打开方式
目前Repo中存储的是程序的全部版本，请到[Releases](https://github.com/DGCK81LNN/hug/releases)页下载最新版本。

对于 **hug2** 从 v2.1.0 (Beta)开始的版本以及 **hug3** 的所有版本，程序运行必须依赖同文件夹下的 hug.json 剧情文件。请先将程序与剧情文件放置在同一个文件夹，然后再运行。
